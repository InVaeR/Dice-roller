# 🎲 Dice Roller

Бросок кубиков с красивыми анимациями и звуковыми эффектами.

## Возможности

- **Режим «Проверка»** — бросок D20 с учётом модификатора и сложности (DC)
  - Автоматическое определение успеха / провала
  - Критический успех (20) и критический провал (1)
  - Кольцо прогресса DC с цветовой индикацией
- **Режим «Бросок»** — свободный бросок любого кубика
  - Поддержка: **D4, D6, D8, D10, D12, D20, D100**
  - Уникальная анимация для каждого кубика:
    - Вращение (rolling) — D6, D8, D12, D20
    - Покачивание (rocking) — D4, D10, D100
- **Выбор навыка** — выпадающий список с 6 характеристиками и 17 навыками D&D
  - Возможность ручного ввода произвольного названия
- **Модальное окно настроек**
  - Кнопка ⚙ в правом нижнем углу экрана
  - Закрытие: кнопка ✕, клик по затемнённому фону, или `Escape`
- **Сохранение модификаторов** — каждый навык запоминает свой модификатор
  - Модификатор отображается прямо в списке: «Сила (+3)»
  - Данные сохраняются между сессиями (localStorage)
  - Кнопка «Сбросить модификаторы» для очистки всех данных
- **Регулировка громкости** — слайдер 0-100% в настройках
- **Визуальные эффекты**: вспышки, искры, руны, пульсация
- **Звуковые эффекты** через Web Audio API
- **История** последних 10 бросков с типом кубика
- **Горячие клавиши**: `Пробел` / `Enter` — бросок, `Escape` — закрыть настройки
