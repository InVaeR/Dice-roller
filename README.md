# D20 Roller

Бросок кубика c красивыми анимациями.

## Возможности

- **Проверка** — бросок с учётом модификатора и сложности (DC), автоматическое определение успеха/провала
- **Свободный бросок** — простой бросок D20 без дополнительных параметров
- Настраиваемый навык, модификатор и DC
- Анимация вращения кубика со звуковыми эффектами
- Визуальные эффекты: вспышки, искры, руны
- История последних 10 бросков
- Горячие клавиши: `Пробел` или `Enter`

## Использование

Откройте `index.html` в браузере.
